## Quick Start

To create your first documentation project with Midhun Gk:

1. Sign in at `https://docs.midhungk.com`.
2. Click "Create Project" on the dashboard.
3. Enter your project name and description.
4. Click "Save" to finish setup.

Your new project will now appear in your dashboard. 
Start adding documentation pages using the sidebar menu.
