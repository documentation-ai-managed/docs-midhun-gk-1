## Upgrade to the Latest Version

Follow these steps to update your Midhun Gk instance and benefit from the newest features.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your documentation space.

    ```bash
    midhun-gk backup --path ./backups/latest
    ```

  </Step>
  <Step title="Update Package" icon="download">

    ```bash
    npm update midhun-gk@latest
    ```

    Or use Yarn:

    ````bash
    yarn upgrade midhun-gk@latest
    ````

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply database migrations if applicable.

    ```bash
    midhun-gk migrate
    ```

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart your Midhun Gk server.

    ```bash
    systemctl restart midhun-gk
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Check the [GitHub repository](https://github.com/midhungk/midhun-gk) for detailed migration guides and community discussions.
</Callout>
