## Overview

Midhun Gk provides your default documentation space to organize and manage all project documentation effortlessly. You create structured docs, collaborate with teams, and integrate with APIs for dynamic content. Start building knowledge bases that scale with your projects using intuitive tools and powerful features.

<Callout kind="info">
Midhun Gk uses brand color `#3B82F6` for primary elements. Apply it in CSS as `color: #3B82F6;` or `--primary: #3B82F6;`.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` with your email.
  </Step>
  <Step title="Set Up Space" icon="settings">
    Choose a template and customize your brand color to `#3B82F6`.
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Click "New Page" and start writing with MDX components.
  </Step>
  <Step title="Invite Team" icon="mail">
    Share your space link: `https://docs.example.com/your-space`.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs space
  const response = await fetch('https://api.example.com/spaces', {
    headers: { Authorization: `Bearer YOUR_API_KEY` }
  });
  const spaces = await response.json();
  console.log(spaces);
  ```
  ```python
  import requests
  response = requests.get(
      'https://api.example.com/spaces',
      headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  spaces = response.json()
  print(spaces)
  ```
</CodeGroup>
